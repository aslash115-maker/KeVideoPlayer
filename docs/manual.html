🎬 播放功能

  • 播放控制：播放/暂停、首末帧跳转、逐帧步进
  • 速度调节：0.25× / 0.5× / 0.75× / 1× / 1.25× / 1.5× / 2× / 4×
  • A-B 循环：标记起止点，自动循环播放区间
  • 音量控制：滑条调节 + 静音切换
  • 全屏模式：F 键或按钮一键进入
  • 拖拽打开：直接把视频拖到窗口

🎨 滤镜系统

  • 33 个滤镜，分 7 类：基础/色彩/黑白/风格化/模糊锐化/失真/氛围
  • 双轨渲染：15 个 LIVE 滤镜播放中实时显示，18 个像素级滤镜暂停后应用
  • 强度滑条：0–100% 平滑混合，所见即所得
  • 瓷砖预览：每个滤镜显示当前画面的预览效果
  • 导出帧：PNG 滤镜效果或原始帧 JPG

🖍 截图与涂鸦

  • 截图：顶栏一键截取当前帧，保存为 PNG（自动合成涂鸦层）
  • 四种工具：画笔自由绘 · 文字标注 · 直线箭头 · 移动已有元素
  • 箭头：按下拖拽出指向终点的三角箭头，头尺寸随线宽自适应
  • 移动：切到 ✥ 工具后悬停命中变 grab，可拖动任意笔画/箭头/文字
  • 独立字号：文字工具激活时显示像素字号输入（8–200），与线宽解耦
  • 颜色与线宽：取色器 + 1–40 px 滑条，所有新增元素沿用当前设定
  • 撤销 / 清除：栈式撤销最近一笔，一键清空全部

🌍 其它

  • 12 种语言：中（简/繁）/英/日/韩/西/法/德/俄/葡/意/阿
  • RTL 支持：阿拉伯语自动镜像布局
  • 帧提取：FFmpeg 高质量 JPEG，可设采样率/上限
  • 胶片条：提取后底部窗口化缩略图导航
  • 键盘速查：按 ? 调出快捷键全表
